# 📦 Sistema de Gestión de Inventario

Este es un sistema de gestión de inventario profesional desarrollado en **Python**. El proyecto implementa una arquitectura modular para separar la lógica de negocio de la interfaz de usuario, permitiendo un control eficiente de productos, existencias y análisis de datos mediante consola.

## ✨ Características Principales

* **Operaciones CRUD completas:** Registro, lectura, búsqueda y actualización de productos.
* **Búsqueda Inteligente:** Motor de búsqueda que ignora diferencias de capitalización (mayúsculas/minúsculas) y espacios en blanco accidentales.
* **Módulo de Estadísticas:** * Cálculo automático de unidades totales.
    * Suma del valor total del inventario.
    * Detección del producto con mayor precio unitario.
    * Identificación del producto con mayor stock disponible.
* **Lógica Optimizada:** Uso de funciones **Lambda** para el procesamiento dinámico de subtotales.
* **Interfaz Profesional:** Diseño de tablas y menús alineados mediante modificadores de formato (`f-strings`) y uso de emojis para mayor legibilidad.

## 🛠️ Tecnologías Utilizadas

* **Lenguaje:** Python 3.x
* **Estructura:** Diccionarios y listas para el manejo de datos en tiempo de ejecución.
* **Control de Versiones:** Git (Rama `main`).

## 🚀 Cómo ejecutar el programa

Para iniciar el sistema de inventario, asegúrate de estar en la carpeta raíz del proyecto y ejecuta el siguiente comando en tu terminal:

python3 main.py
